# Cryato

《Cryato》 是一款移动应用程序，可使加密货币套利计算变得简单。这是一款离线应用程序，
允许用户在考虑最新的汇率和市场条件的情况下，计算从不同交易所购买和出售不同加密货币所获利润。
该应用程序旨在简单易用，不需要任何交易或加密货币套利方面的先前知识。通过《Cryato》，用户可以
做出明智的投资决策，最大化从加密货币套利中获得的利润。
